## Версия ядра 1.0.0 нестабильная и не рекомендуется для использования в производстве. Она предназначена только для тестирования и разработки.

### Документация по разработке здесь: [dev/StartCore.md](dev/StartCore.md)

## Дорожная карта

Версия 1.0.0 является начальным выпуском ядра analog-wakatime-lite-core. Для этой версии запланированы следующие функции:

- [x] Основная функциональность (Базовое отслеживание и логирование активности кодирования)
- [x] Интеграция API (Интеграция с API Analog WakaTime для синхронизации данных)
- [x] Аутентификация пользователей (Базовый механизм аутентификации для пользователей)
- [x] Хранение данных (Локальное хранилище для отслеживания данных)
- [x] Пользовательский интерфейс (Базовый UI для отображения отслеживаемых данных)
- [x] Обработка ошибок (Базовые механизмы обработки ошибок и логирования)
- [x] Оптимизация производительности (Оптимизация производительности ядра для лучшей точности отслеживания)
- [x] Документация (Полная документация для разработчиков и пользователей )
- [x] Тестирование (Модульное и интеграционное тестирование для основной функциональности)
- [x] Обратная связь и итерация (Сбор отзывов пользователей и итерация функций на основе отзывов)
- [x] Интег с третьими сторонами (Интеграция с популярными редакторами кода и IDE для бесшовного отслеживания)
- [x] Кроссплатформенная поддержка (Обеспечение совместимости с несколькими операционными системами)
- [x] Улучшения безопасности (Реализация мер безопасности для защиты данных пользователей)
