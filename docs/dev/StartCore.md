# To work with the Сore you need

---

1. Install the latest version of GO Version 1.26.0 (<https://go.dev/dl/>).
2. Clone the repository to your local machine using the following command:

```bash
git clone https://github.com/analog-wakatime/analog-wakatime-lite-core.git
```

3. Navigate to the project directory:

```bash
cd analog-wakatime-lite-core
```

4. Install the required dependencies using the following command:

```bash
go mod tidy
```

5. Build the project using the following command:

```bash
go build -o analog-wakatime-lite-core
```

6. Run the core using the following command:

```bash
./analog-wakatime-lite-core
```
